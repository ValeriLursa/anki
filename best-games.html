<a> - Тег для добавления ссылки на другой сайт или ссылки-якоря Словарь
<h> - Тег для добавления заголовка, существует 6 видов, от h1 до h6 Словарь
text-decoration - Свойство текста, добавляет стиль в виде подчеркивания Словарь
text-align - Свойство текста, выравнивает текст по горизонтали в пределах элемента Словарь
text-shadow - Свойство текста, добавляет тень вокруг слов.
Содержит 4 значения: сдвиг по горизонтали (px) сдвиг по вертикали (px) радиус размытия (px) цвет (rgb, hex) 		Словарь
RGB - Задает цвет используя значения зеленого, красного и синего, в формате трех чисел rgb(255, 255, 255). Значения каждого параметра могут быть от 0 до 255 Словарь
HEX - Задает цвет, используя 6 цифр. Формат: #000000 - черный. Каждая цифра может иметь значение от 0 до 9 и от A до F (#FFFFFF - белый) Словарь

border - Общее свойство, добавляет рамку вокруг элемента. Содержит 3 значения: толщина стиль цвет Словарь
<i> - Тег для выделения текста курсивным начертанием Словарь
<b> - Тег для выделения текст жирным начертанием Словарь
class - Атрибут для добавления уникальных свойств тегу Словарь
border-radius - Свойство для округления углов у элемента. Может содержать одно, два и четыре значения (px) Словарь
id - Атрибут для добавления уникальных свойств тегу и используется для ссылок-якорей. Значение должно быть уникально на всей веб-странице и использоваться один раз Словарь
box-shadow - Свойство используется для добавления тени элементу.
Содержит 4 значения: сдвиг по горизонтали (px) сдвиг по вертикали (px) радиус размытия (px) цвет (rgb, hex)		 Словарь

Может содержать одно, два и четыре значения в px: свверху, справа, снизу, слева. 		Словарь
padding - Общее свойство, добавляет отступ от контента внутри элемента до границы элемента.
Может содержать одно, два и четыре значения в px: свверху, справа, снизу, слева. 		Словарь
