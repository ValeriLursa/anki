Теория
float - определяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон. Словарь
opacity - определяет уровень прозрачности элемента веб-страницы Словарь
overflow - свойство управляет отображением содержания блочного элемента, если оно целиком не помещается и выходит за область заданных размеров Словарь
line-height - Устанавливает межстрочный интервал текста Словарь
transform - Трансформирует элемент, в частности, позволяет его масштабировать, вращать, сдвигать, наклонять, а также комбинировать виды трансформаций Словарь
transform: translateX() / translateY() - Сдвигает элемент по горизонтали или по вертикали на указанное значение Словарь
transform: translate() - Общее свойство, сдвигает элемент по горизонтали и по вертикали на указанное значение Словарь
transform: scaleX() / scaleY() - Масштабирует элемент по горизонтали или по вертикали, значение больше 1 - увеличивает элемент, а значение меньше 1 - уменьшает Словарь
transform: scale() - Общее свойство, масштабирует элемент по горизонтали и по вертикали Словарь
transition - Устанавливает эффект перехода между двумя состояниями элемента Словарь
:hover - Псевдо-класс, определяет стиль элемента при наведении на него курсора мыши, но при этом элемент еще не активирован, иными словами кнопка мыши не нажата Словарь
@keyframes - Ключевое слово, управляет промежуточными шагами в последовательности анимации CSS, определяя стили для ключевых кадров (или путевых точек) последовательности анимации Словарь
animation-name - Указывает имена одного или нескольких правил, описывающих анимацию, применяемую к элементу Словарь
animation-duration - Задает время, за которое анимация завершает один цикл, проходит от 0% до 100% Словарь
animation-iteration-count - Устанавливает, сколько раз последовательность анимации должна воспроизводиться перед остановкой Словарь
<span> - Тег, предназначен для строк, может выделить текст внутри другого тега для добавления различных стилей Словарь
animation-timing-function - Позволяет настроить как свойства в анимации будут перетекать друг в друга Словарь
filter - Позволяет применить различные графические эффекты на элемент Словарь
