Основные теги и атрибуты

<html> - является контейнером, который заключает в себе всё содержимое веб-страницы Словарь
<head> - предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными Словарь
<title> - Определяет заголовок вкладки сайта Словарь
<body> - предназначен для хранения содержания веб-страницы (контента), отображаемого в окне браузера Словарь
<h[1-6]> - заголовки разного уровня важности. По умолчанию отображаются крупно и с жирным начертанием Словарь
<img> - предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG Словарь
<img src="URL"> - атрибут, предназначеный для добавления адреса на файл с картинкой Словарь
<img width="100px"> - атрибут для изменения ширины картинки Словарь
<p> - определяет текстовый абзац. Является блочным элементом, всегда начинается с новой строки. Абзацы текста идущие друг за другом разделяются между собой отбивкой, пустым промежутком Словарь
<br> - устанавливает перенос строки в том месте, где этот тег находится Словарь
<a> - тег для создания ссылок Словарь
<a href="URL"> - атрибут, которыйпередает адрес для ссылки, на которую нужно перейти Словарь
<iframe> - позволяет добавить видео на сайт с другого сайта Словарь
<link> - устанавливает связь с внешним документом вроде файла со стилями или со шрифтами Словарь
