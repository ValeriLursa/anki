Сайт "Комикс"

background-image - Свойство для добавления изображения на задний фон элемента Словарь
background-size - Свойство, которое изменяет размеры изображения на фоне элемента Словарь
background-position - Свойство, которое задает положение изобраения на фоне в элементе Словарь
background-attachment - Свойство, которое устанавливает, прокручивается ли фоновое изображение вместе с содержимым элемента Словарь
background-repeat - Свойство, которое устанавливает, будет ли повторяться фоновое изображение, если его размеры меньше элемента Словарь
<hr> - Тег-разделитель, добавлять линию на страницу Словарь
@import - Ключевое слово для подключения стиля текста из интернета по ссылке Словарь
