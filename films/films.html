Теория
<ol> - устанавливает нумерованный список Словарь
<ul> - устанавливает маркированный список, от слова маркер Словарь
<li> - определяет отдельный элемент списка Словарь
background-color - определяет цвет фона элементаа Словарь
color - определяет цвет текста элемента Словарь
list-style-type - изменяет вид маркера для каждого элемента списка Словарь
RGB - задает цвет используя значения зеленого, красного и синего, в формате трех чисел rgb(255, 255, 255). Значения каждого параметра могут быть от 0 до 255 Словарь
HEX - задает цвет, используя 6 цифр. Формат: #000000 - черный. Каждая цифра может иметь значение от 0 до 9 и от A до F (#FFFFFF - белый) Словарь
<тег class="имя класса"> - атрибут, задающий стили конкретному элементу. Разрешается одному элементу присваивать несколько классов через пробел, так и один класс нескольким элементам Словарь
font-size - свойство, определяет размер шрифта элемента Словарь
text-decoration - свойство, добавляет оформление текста в виде его подчеркивания, перечеркивания, линии над текстом и мигания. Одновременно можно применить более одного стиля, перечисляя значения через пробел Словарь
text-align - свойство, определяет горизонтальное выравнивание текста в пределах элемента Словарь
text-shadow - свойство, добавляет тень к тексту.
Содержит 4 значения: сдвиг по горизонтали (px) сдвиг по вертикали (px) радиус размытия (px) цвет (rgb, hex)		 Словарь
font-family - свойство, устанавливает семейство шрифта, которое будет использоваться для оформления текста содержимого Словарь
@import - правило, позволяет импортировать содержимое CSS-файла в текущую стилевую таблицу Словарь
