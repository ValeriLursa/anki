Теория
<img> - предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG Словарь
<img src="URL"> - атрибут, предназначеный для добавления адреса на файл с картинкой Словарь
<img alt="текст"> - атрибут, предназначеный для добавления альтернативного текста для картинки Словарь
<p> - определяет текстовый абзац. Является блочным элементом, всегда начинается с новой строки. Абзацы текста идущие друг за другом разделяются между собой отбивкой, пустым промежутком Словарь
<a> - тег для создания ссылок Словарь
<a href="URL"> - атрибут, которыйпередает адрес для ссылки, на которую нужно перейти Словарь
<h[1-6]> - заголовки разного уровня важности. По умолчанию отображаются крупно и с жирным начертанием Словарь
text-decoration - свойство, добавляет оформление текста в виде его подчеркивания, перечеркивания, линии над текстом и мигания. Одновременно можно применить более одного стиля, перечисляя значения через пробел Словарь
text-align - свойство, определяет горизонтальное выравнивание текста в пределах элемента Словарь
text-shadow - свойство, добавляет тень к тексту.
Содержит 4 значения: сдвиг по горизонтали (px) сдвиг по вертикали (px) радиус размытия (px) цвет (rgb, hex)		 Словарь
<div> - тег-контейнер для объединения других тегов, блочный элемент Словарь
border - общее свойство, добавляет рамку вокруг элемента. Содержит 3 значения: толщина стиль цвет Словарь
<i> - тег для выделения текста курсивным начертанием Словарь
<b> - тег для выделения текст жирным начертанием Словарь
class - атрибут для добавления уникальных свойств тегу Словарь
border-radius - свойство для округления углов у элемента. Может содержать одно, два и четыре значения (px) Словарь
<тег id="текст"> - атрибут для добавления уникальных свойств тегу и используется для ссылок-якорей. Значение должно быть уникально на всей веб-странице и использоваться один раз Словарь
box-shadow - свойство используется для добавления тени элементу.
Содержит 4 значения: сдвиг по горизонтали (px) сдвиг по вертикали (px) радиус размытия (px) цвет (rgb, hex)		 Словарь
margin - общее свойство, добавляет отступ от границы элемента до других элементов.
Может содержать одно, два и четыре значения в px: свверху, справа, снизу, слева. 		Словарь
padding - общее свойство, добавляет отступ от контента внутри элемента до границы элемента.
Может содержать одно, два и четыре значения в px: свверху, справа, снизу, слева. 		Словарь
vertical-align - свойство для выравнивания элемента по вертикали относительно своего родителя Словарь
